# Project conventions (auto-loaded instructions)

## Response prefix — [Graphify]

If you used graphify in any way while doing the work for a reply — running
`graphify query` / `path` / `explain` / `update`, or reading anything under
`graphify-out/` (graph.json, GRAPH_REPORT.md, the wiki) — then begin that
reply with the literal token `[Graphify]` as the very first characters,
before anything else.

- One prefix per reply, at the start only.
- If you did NOT use graphify for that reply, do not add the prefix.

This makes it obvious at a glance which answers were informed by the
knowledge graph. (See the graphify usage rules in the parent CLAUDE.md.)
